# Template Forge Pro

코딩을 모르는 사용자도 실제 배포 가능한 홈페이지를 만들 수 있는 노코드 템플릿 생성기입니다.

## 구현된 기능

- 메인 페이지 꾸미기
  - 이미지 첨부(URL/파일 업로드)
  - 영상 첨부(URL/파일 업로드, 최대 20MB)
  - 텍스트/버튼 링크 변경
  - 섹션별 레이아웃(좌우/상단/하단/커버), 미디어 위치(좌/우/위/아래/덮기)
  - 섹션별 컬러/폰트, 간단 효과(페이드/슬라이드/줌/없음)
- 1섹션, 2섹션 꾸미기
  - 이미지/영상/텍스트/레이아웃/컬러/폰트/효과 동일 지원
- 추가 섹션 무제한 추가/삭제
- 텍스트 링크 삽입
  - 본문에 `[링크텍스트](https://example.com)` 형식 입력 가능
- 풋터 꾸미기
  - 전화번호, 주소, 계약서 링크, 개인정보처리방침 링크
- 공통 기능
  - 레이아웃 추천(브랜드 소개형/제품 강조형/스토리형)
  - 컬러 변경(대표/배경)
  - 무료 폰트 선택(Pretendard, Noto Sans KR)
  - 스크롤 애니메이션(IntersectionObserver 기반)
- 배포/검색 대응
  - `index.html`, `robots.txt`, `sitemap.xml` 자동 생성 다운로드
  - SEO 메타 태그, Open Graph, JSON-LD 포함

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 왼쪽 편집기에서 메인/섹션/풋터를 수정합니다.
3. `배포 파일 다운로드`를 누릅니다.
4. 생성된 파일 3개를 호스팅에 업로드합니다.

## 배포 예시

- GitHub Pages
- Netlify
- Vercel

배포 도메인이 정해지면 `도메인 URL` 값을 실제 주소로 바꾼 뒤 다시 다운로드해서 재배포하세요.

## 참고

- 이미지 파일 업로드 제한: 최대 5MB
- 영상 파일 업로드 제한: 최대 20MB
- 대용량 영상은 외부 CDN/스토리지 URL 사용 권장
